# Banza Village Site

PHP/SQL website project.

## Project Layout

- `SITE/` - website source code.
- `AGENTS.md`, `.agents/`, `.codex/`, `docs/`, `templates/`, `scripts/` - Codex starter pack tools and guidance, not product code.

## Local Development

Use XAMPP while building the PHP/SQL site locally.

Typical local URL:

```text
http://localhost/SITE/
```

GitHub stores source code and project history. GitHub Pages will not run PHP or SQL.
