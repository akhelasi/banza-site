# Banza Seed Content Sources

Seed content for the first build was summarized and rewritten in Georgian from the sources below. Some homepage items are intentionally demo content for layout and QA until the client provides official text, current figures, bank accounts, live camera details, weather API settings, and project/news records.

## Facts Used In Seed Content

- ბანძა არის სოფელი მარტვილის მუნიციპალიტეტში, სამეგრელო-ზემო სვანეთის მხარეში.
- სოფელი მდებარეობს ოდიშ-გურიის დაბლობზე, მდინარე აბაშის მარცხენა სანაპიროზე.
- ზღვის დონიდან სიმაღლე წყაროებში მითითებულია 90 მეტრად.
- 2014 წლის აღწერის მონაცემით მოსახლეობა მითითებულია 1099 ადამიანად.
- ქართული Wikipedia-ს მიხედვით ბანძა პირველად მოხსენიებულია 1639-1640 წლებში.
- წყაროებში ნახსენებია ბანძის ღვთისმშობლის მიძინების ტაძარი, ბანძის სინაგოგა და სოფლის ისტორიული კონტექსტი.
